The <nx-sidebar> provides a collapsible container for side content (often navigation, though it can be any content) alongside some primary content.

Please note that all the following examples use the NxActionComponent for the side navigation items. The NxActionModule is not automatically imported when you import the NxSidebarModule. Therefore you have to import the NxActionModule yourself, if you want the same sidebar look:

import { NxActionModule } from '@aposin/ng-aquila/action';
