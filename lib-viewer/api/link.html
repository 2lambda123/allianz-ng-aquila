Directives

NxLinkComponent extends MappedStyles

Input('nxStyle') classNames sets the style of the link, thereby altering the visual appearance. You can use any combination of 'black', 'icon-right', 'icon-only', 'negative' or 'text'.

Selector:

nx-link

Properties
Name Description
@Input()

size: NxLinkSize

Sets the size of the link. Default: 'small'.

Additional type aliases

NxLinkSize

The size of the link.

Type: 'small' | 'large'
