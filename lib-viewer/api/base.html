Services

ErrorDefaultOptions

Represents the default options for the error notification that can be configured using the ERROR_DEFAULT_OPTIONS injection token.

Properties
Name Description

appearance: ErrorStyleType

Defines the style type of the error notification.

changes: Subject

Stream that emits whenever the default options are changed. Use this to notify components if the default options have changed after initialization.

Directives

nx-error

Properties
Name Description
@Input()

appearance: ErrorStyleType

Whether the error should have message or text styling.

Default is 'message'.
@Input()

id: string

Id of the nx-error.

If not set, the selectable card gets an incremented value by default.
@Input()

showIcon: boolean

Whether an icon should be displayed. Only has an effect for type 'text'
