NxFormfieldComponent

Selector:

nx-formfield

Properties
Name Description
@Input()

appearance: AppearanceType

Expert option

Sets the appearance of the formfield.
@Input('nxFloatLabel')

floatLabel: FloatLabelType

Whether the label should float once the input is focused or filled (auto, default) or force it to always float with a value of always to simulate a more static form.
@Input('nxLabel')

label: string

Sets the label which will act as a floating label. In addition, the component uses input and label to properly support accessibility.
@Input('nxStyle')

styles: string

Sets the styling of the formfield. If 'negative', a negative set of stylings is used.

labelId: string

Html id of the formfield label

Additional interfaces

FormfieldDefaultOptions

Represents the default options for the form field that can be configured using the FORMFIELD_DEFAULT_OPTIONS injection token.

Properties
Name Description

appearance: AppearanceType

Sets the default appearance. (optional)

nxFloatLabel: FloatLabelType

Sets the default float label type. (optional)

Additional type aliases

FloatLabelType

Type for the available floatLabel values.

Type: 'always' | 'auto'

AppearanceType

Type for the appearance of the formfield.

Type: 'outline' | 'auto'
