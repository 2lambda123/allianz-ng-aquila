Directives

NxDataDisplayLabelComponent

Label within a <nx-data-display>.

Selector:

nx-data-display-label

NxDataDisplayComponent

Data display component.

Selector:

nx-data-display

Properties
Name Description
@Input()

label: string | null

Sets the label which will describe the data.
@Input()

orientation: NxDataDisplayOrientation

Layout orientation of the label and value.

  • 'vertical': label and value are vertically stacked.
  • 'horizontal': label and value are on the same line.
@Input()

size: NxDataDisplaySize

Size of the data display.

Additional interfaces

DataDisplayDefaultOptions

Default options for the NxDataDisplayComponent.

Properties
Name Description

size: NxDataDisplaySize

Additional type aliases

NxDataDisplayOrientation

Layout orientation of the NxDataDisplayComponent. 'vertical': label and value are vertically stacked. 'horizontal': label and value are on the same line.

Type: 'vertical' | 'horizontal'

NxDataDisplaySize

Sizes of the NxDataDisplayComponent.

Type: 'small' | 'medium' | 'large'
