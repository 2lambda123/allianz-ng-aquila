NxMaskDirective

Selector:

input[nxMask]

Exported as: nxMaskDirective
Properties
Name Description
@Input('nxConvertTo')

convertTo: MaskConversionTypes

Sets the case sensitivity of the mask.
@Input()

deactivateMask: boolean

Whether the mask validation should be deactivated.
@Input()

dropSpecialCharacters: boolean

Whether the separators should be dropped in the control value accessor.
@Input('nxMask')

mask: string

Sets the mask.
@Input()

separators: string[]

Sets the keys that are recognized as separators. Default separators: / ( ) . : - + , and space.
@Input()

validateMask: boolean

Whether the mask validation should be applied on the input. Default: true.

cvaModelChange: Subject

Emits the unmasked value before the value changes.
Methods
getUnmaskedValue(): string

Returns the unmasked value.

registerAfterInputHook(afterInput: (event)

Registers a function to be executed after the onInput handler. The registered hook receives a KeyboardEvent from the onInput event handler as a parameter.

Note: If you register a afterInputHook, you may also register a beforePasteHook to perform similar changes for pasting.

Parameters:

afterInput: (event
registerBeforePasteHook(beforePaste: (event)
